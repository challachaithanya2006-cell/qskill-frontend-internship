# 🚀 QSkill Front-End Internship Project

A modern React web application developed as part of the **QSkill Front-End Development Internship**.

## 🌐 Live Demo

https://qskill-frontend-internship.vercel.app/

## 📂 GitHub Repository

https://github.com/challachaithanya2006-cell/qskill-frontend-internship

---

## ✨ Features

- 🌍 Text Translator using Microsoft Translator API
- 🔐 Random String Generator
- 🏠 Responsive Home Page
- 👨 About Page
- 📱 Responsive Design
- 🚀 React Router Navigation
- ⚡ Fast Vite Build

---

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Router
- Axios
- Microsoft Translator API (RapidAPI)

---

## ⚙️ Installation

```bash
git clone https://github.com/challachaithanya2006-cell/qskill-frontend-internship

cd qskill-frontend-internship

npm install

npm run dev
```
